Detailed Description

A pool allocator for Karma's UObjects. I'd higly recomment Gregory's Game Engine Architecture section 5.2 for introductory level and practical approach to memory system.

A modular memory system https://github.com/ravimohan1991/cppGameMemorySystem Karma's take https://github.com/ravimohan1991/KarmaEngine/wiki/Karma-Smriti
